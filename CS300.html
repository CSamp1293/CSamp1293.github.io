This project was originally developed for my CS 300: Data Structures and Algorithms course. The original program, written in C++, displayed courses and their prerequisites but did not resolve full prerequisite chains. For the CS 499 enhancement, I rewrote the program in Python, implementing recursive logic to trace and display the full chain of prerequisites for any course.
